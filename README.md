Os níveis de dificuldade das tarefas no Codewars são definidos pelos valores “kyu”.
Quanto menor o número, maior a dificuldade.

O sistema vai de 8 kyu (mais fácil) até 1 kyu (mais difícil).

Os níveis 1, 2, 3 e 4 kyu envolvem problemas técnicos complexos, geralmente resolvidos por desenvolvedores sêniores.
Por enquanto, vou focar nos desafios entre 8 kyu e 5 kyu, que são mais adequados ao meu nível atual.
